== Description ==

Automatically remove exif and metadata data after uploading. Just moment supported format: JPG and PNG. Using ImageMagick


== Installation ==

1. Upload the directory 'remove-exif-and-metadata' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload images while writing posts and pages.
